## install ##

From rubygems:  
~~~
    [sudo] gem install num4probstdy
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/probstdy.git  
    cd probstdy  
    gem build *.gemspec
    gem install *.gem
~~~
